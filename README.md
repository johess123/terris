<h1>TetrisXNaruto</h1>
<h2>使用說明</h2>
  1. 打開 dist 資料夾中的 TetrisXNaruto.exe 執行遊戲
  ![](https://hackmd.io/_uploads/BJR2zx782.png)
  2. 可以選擇開始級數
  ![image](https://github.com/johess123/tetris/assets/104426729/2d6e2593-746f-402f-bdf7-b858d322c77b)
<h2>操作方法</h2>
  1. 左 A
  2. 右 D
  3. 下 S
  4. 直接墜落 空白鍵
  5. 逆時鐘旋轉 J
  6. 順時鐘旋轉 K
